# SmartVote Pro

**SmartVote Pro** is a **blockchain-based AI fraud detection voting system** built with **Python** and **Streamlit**.  
It allows users to cast votes, verify votes, explore election data, and detect anomalies in voting using AI.  

## Features

- **Role-based access:** Voter, Admin, Security
- **Voting Portal:** Users can cast votes securely
- **Verify Vote:** Check your vote status
- **Election Dashboard:** Admin can see vote counts with charts (bar and pie)
- **Fraud Detection:** Detect suspicious voting patterns using Isolation Forest
- **Blockchain Explorer:** Track votes in a tamper-proof blockchain

## Technologies Used

- **Python:** Core programming language  
- **Streamlit:** For web interface  
- **Pandas & NumPy:** Data processing  
- **Matplotlib:** Charts & visualization  
- **Scikit-learn:** AI-based anomaly detection
